## Lessons Learned

- [Tooling] Mistake: 批量生成脚本调 API 遇到配额问题 | Pattern: 直接由 Claude 生成 HTML 更可靠 | Rule: 有 Claude 订阅时，优先用 Claude 直接生成而非调 API
- [Architecture] Pattern: subagent 并行生成（5个一批）效率极高，每个实验 3-5 分钟 | Rule: 独立文件任务适合并行 agent
- [Architecture] Pattern: A0 route.ts 迁移暴露了 provider 选择逻辑（zenmux 代理），迁入 generateCore 后双路径统一 | Rule: 安全修复优先于功能开发
