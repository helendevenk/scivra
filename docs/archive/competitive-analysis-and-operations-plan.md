## 第一部分：竞品深度调研

### 一、物理仿真/实验平台

#### 1. PhET Interactive Simulations (phet.colorado.edu)

**产品定位与核心功能**
- 科罗拉多大学博尔德分校开发的免费在线交互式仿真平台
- 覆盖物理、化学、生物、地球科学、数学五大学科
- 提供 160+ 个 HTML5 仿真实验，已从 Java/Flash 全面迁移至浏览器原生技术
- PhET Studio：允许教师自定义仿真参数，创建定制化实验
- 支持 40+ 种语言翻译，全球化覆盖极广
- 提供 iOS/Android 移动端应用，支持离线访问

**商业模式与定价策略**
- 核心仿真完全免费，开源代码
- PhET Studio 采用付费许可模式（免费试用 + 购买许可证）
- 资金来源：Moore 基金会、NSF、Hewlett 基金会、Yidan Prize、Mastercard 基金会等
- 本质是学术项目，非商业化运营

**用户规模与市场地位**
- 全球物理教育仿真领域的绝对标杆，累计使用量超过数十亿次
- 被全球数十万教师采用，几乎所有物理教育资源列表都会推荐 PhET
- 在 Google 搜索 "physics simulation" 中长期占据前三

**内容策略与获客方式**
- 依托科罗拉多大学学术品牌背书
- 教师社区口碑传播为主
- 提供教学活动设计、虚拟工作坊、使用技巧等教师资源
- SEO 自然流量极强

**技术特点与体验**
- HTML5 Canvas/SVG 为主，2D 交互为核心
- 无障碍设计（Accessibility）做得很好，支持屏幕阅读器
- 仿真质量高但视觉风格偏学术/简朴
- 无 AI 能力，无 3D 场景

**相对于 SeePhysics 的优劣势**
- 优势：免费、学术权威、海量仿真、全球知名度、多语言
- 劣势：无 AI 生成能力、无 3D 可视化、无法按需生成自定义场景、视觉体验较旧、无公式推导整合

#### 2. The Physics Classroom (physicsclassroom.com)

**产品定位与核心功能**
- 面向学生和教师的综合物理教育网站，"由教师为教师和学生编写"
- 核心内容：物理教程（文字版在线教科书）、视频教程、复习模块
- 交互工具：近 100 个物理仿真、Concept Builders（概念构建器）、Minds On Physics（151 个智能问答任务）
- 辅助工具：Calc Pad（计算练习）、Science Reasoning（科学推理练习，对齐 NGSS 和 ACT）
- ACT 考试准备中心

**商业模式与定价策略**
- 免费增值模式（Freemium）
- 基础教程和部分交互工具免费但有广告
- 付费产品：无广告计划、Task Tracker（教师追踪学生进度）、Solutions Guides、Physics Interactive Pack、Conceptual Physics Course Pack、Teacher Presentation Pack、Question Bank
- 支持机构批量采购

**用户规模与市场地位**
- 北美物理教育领域知名度极高，尤其在高中物理教师群体中
- 内容全面性在同类网站中数一数二

**内容策略与获客方式**
- 海量 SEO 友好的文字教程内容，长尾关键词覆盖极广
- 教师社区推荐和口碑传播
- 与标准化考试（ACT）对齐吸引应试需求

**技术特点与体验**
- JavaScript 重度 Web 应用
- 仿真为 2D HTML5/Canvas
- 有用户账户系统、明暗模式切换
- 广告较多（使用 Freestar 广告平台）

**相对于 SeePhysics 的优劣势**
- 优势：内容极其全面（教程+仿真+练习+考试准备）、SEO 流量大、教师工具成熟
- 劣势：无 AI、无 3D、视觉体验一般、广告干扰体验、无中文支持、无法生成自定义内容

#### 3. oPhysics (ophysics.com)

**产品定位与核心功能**
- 由退休 AP 物理教师 Tom Walsh（27 年教学经验）创建的交互式物理仿真集合
- 120+ 个仿真实验，覆盖运动学、力学、守恒定律、波动、光学、电磁学、转动、流体、现代物理
- 附带绘图工具（电路图、力学图、图表生成器）
- 趣味内容区（光学错觉、日食探索等）

**商业模式与定价策略**
- 完全免费，接受 PayPal 捐赠
- 仅限非营利教育用途

**用户规模与市场地位**
- AP 物理教师群体中有一定知名度
- 规模远小于 PhET，属于个人项目

**技术特点与体验**
- 基于 GeoGebra 构建所有仿真
- 2D 交互，视觉风格简洁但功能性强
- 无移动端优化

**相对于 SeePhysics 的优劣势**
- 优势：免费、AP 物理覆盖全面、教师视角设计
- 劣势：个人项目维护风险、无 AI、无 3D、无移动端、无法扩展、依赖 GeoGebra

#### 4. Falstad Circuit Simulator (falstad.com)

**产品定位与核心功能**
- Paul Falstad 个人开发的浏览器端电路仿真器及物理/数学 Applet 集合
- 核心产品：Circuit Simulator（电路仿真器）——电子工程社区广泛使用
- 附带：波动仿真、膜振动仿真、迷宫等物理/数学可视化
- 提供 iOS/Android 移动应用

**商业模式与定价策略**
- 完全免费，开源
- 无商业化运营

**用户规模与市场地位**
- 电路仿真领域知名度很高，被电子工程爱好者和学生广泛使用
- 物理仿真部分影响力有限

**技术特点与体验**
- 浏览器端 JavaScript 运行
- 电路仿真器交互体验优秀，实时可视化电流/电压
- 界面风格极简/复古

**相对于 SeePhysics 的优劣势**
- 优势：电路仿真领域专业度极高、免费开源
- 劣势：仅覆盖电路/电磁领域、无 AI、无 3D、界面老旧、无教学内容整合

#### 5. Algodoo

**产品定位与核心功能**
- Algoryx 开发的 2D 物理沙盒仿真软件（前身为 Phun）
- 用户通过绘图工具创建物理场景，实时观察物理行为
- 覆盖：重力、摩擦力、浮力、刚体动力学、光学（透镜和激光）、齿轮和马达等
- Algobox 社区平台：用户上传和浏览创作（车辆、火箭、月球着陆器等）

**商业模式与定价策略**
- 现已完全免费（此前为付费软件）
- 接受捐赠支持开发

**用户规模与市场地位**
- 在物理教育沙盒领域有独特地位
- 用户群体偏向创意探索和趣味学习
- 社区活跃度中等

**技术特点与体验**
- 桌面应用（Windows/Mac，支持 Apple Silicon）+ iPad
- 2D 物理引擎，绘图式交互
- 可视化辅助：颜色、图表、力的显示
- 非浏览器端，需要安装

**相对于 SeePhysics 的优劣势**
- 优势：沙盒自由度极高、创意性强、社区分享
- 劣势：仅 2D、需安装、无 AI、无公式推导、无课标对齐、无 Web 端

### 二、AI 教育工具

#### 6. Khan Academy + Khanmigo AI

**产品定位与核心功能**
- 全球最大的免费在线教育平台之一，覆盖数学、科学、计算机、人文等全学科
- Khanmigo：基于 GPT-4 的 AI 辅导助手（2023 年推出）
  - 苏格拉底式对话教学：不直接给答案，引导学生思考
  - AI 辅导：逐步引导解题过程
  - 教师工具：课程计划生成、学生进度分析
  - 写作辅导、辩论练习等
- 物理课程：从基础物理到 AP 物理，包含视频、练习、文章

**商业模式与定价策略**
- Khan Academy 核心内容完全免费
- Khanmigo：
  - 个人用户：$44/年（约 $3.67/月）
  - 学区/机构：按学生数量定价
  - 2024 年起逐步向更多用户开放
- 资金来源：Bill & Melinda Gates 基金会、Google.org 等慈善捐赠

**用户规模与市场地位**
- 全球 1.5 亿+ 注册用户
- 教育科技领域的绝对巨头
- Khanmigo 用户快速增长中

**内容策略与获客方式**
- 海量免费优质内容驱动自然增长
- Sal Khan 个人品牌影响力
- 与学校/学区深度合作
- YouTube 频道（数千万订阅）

**技术特点与体验**
- Web + 移动端全平台
- AI 对话式交互（Khanmigo）
- 视频 + 练习 + 文章的混合学习模式
- 无物理仿真/3D 可视化

**相对于 SeePhysics 的优劣势**
- 优势：免费、海量内容、AI 辅导、全球知名度、全学科覆盖
- 劣势：无 3D 物理仿真、无可视化生成、物理内容以视频/文字为主缺乏交互性、Khanmigo 是对话式而非可视化式

#### 7. Brilliant.org

**产品定位与核心功能**
- "Learn by doing" 交互式 STEM 学习平台
- 覆盖数学、计算机科学、数据科学、自然科学四大领域
- 核心特色：交互式问题解决（非视频教学）
  - 逐步交互式课程，实时智能反馈
  - 个性化学习路径，追踪掌握程度，自适应节奏
  - 引导式课程，渐进式构建问题解决能力
- 60+ 门结构化课程

**商业模式与定价策略**
- 免费增值模式
- 免费：有限课程内容和每日挑战
- Premium：
  - 月付约 $24.99/月
  - 年付约 $149.99/年（约 $12.50/月）
  - 解锁全部课程和功能
- 家庭/团队计划可用

**用户规模与市场地位**
- 1000 万+ 全球学习者
- 10 万+ 五星应用评价
- 被 NYT、The Atlantic 等主流媒体推荐
- STEM 交互学习领域的领导者

**内容策略与获客方式**
- 高质量交互内容驱动口碑
- 社交媒体广告投放（YouTube、Instagram 等）
- 每日挑战题吸引用户回访
- 应用商店优化

**技术特点与体验**
- Web + iOS + Android 全平台
- 自研交互式课程引擎
- 精美的视觉设计和动画
- 无物理仿真引擎，交互以问答和拖拽为主

**相对于 SeePhysics 的优劣势**
- 优势：交互设计精美、课程体系完整、个性化学习、品牌知名度高
- 劣势：无物理仿真/3D 可视化、无 AI 生成能力、物理内容深度有限（偏概念而非实验）、价格较高、无中文

#### 8. Wolfram Alpha

**产品定位与核心功能**
- 计算知识引擎，"使用 Wolfram 突破性算法、知识库和 AI 技术计算专家级答案"
- 基于 Wolfram Language，35+ 年研发积累
- 四大核心能力：自然语言理解、策展数据与知识、动态算法计算、计算可视化
- 物理能力：可解决力学、电磁学、热力学、量子力学等各类物理计算问题
- Step-by-Step Solutions：逐步展示解题过程（Pro 功能）
- 提供 API 和 LLM 集成方案

**商业模式与定价策略**
- 三层定价：
  - Basic（免费）：基础计算，无逐步解答
  - Pro（$5-9.99/月）：逐步解答、练习题、2x 计算时间、2MB 上传、可下载图形
  - Pro Premium（$8.25-12/月）：3x 计算时间、5MB 上传、优先支持
- 学生/教育者特别定价（学生有学期付费选项）

**用户规模与市场地位**
- 计算引擎领域无可争议的领导者
- 被全球学生、教师、研究者广泛使用
- 已集成到 ChatGPT、Siri 等主流 AI 产品中

**内容策略与获客方式**
- 搜索引擎自然流量（"how to solve..." 类查询）
- 与 AI 平台集成扩大触达
- 学术社区口碑
- Wolfram 生态系统（Mathematica、Wolfram Language）协同

**技术特点与体验**
- 强大的符号计算和数值计算引擎
- 自然语言输入
- 2D 图表和可视化输出
- 无 3D 交互仿真

**相对于 SeePhysics 的优劣势**
- 优势：计算能力无与伦比、逐步解答、覆盖全学科、API 生态
- 劣势：纯计算工具而非学习/探索平台、无 3D 可视化、无交互式仿真、界面偏工具化不够友好、无课标对齐

### 三、科学可视化工具

#### 9. Desmos

**产品定位与核心功能**
- "Beautiful free math" —— 免费数学可视化工具
- 核心产品：
  - 图形计算器：交互式函数图形绘制
  - 科学计算器
  - 几何工具
  - Desmos Classroom（现已并入 Amplify）：教师创建交互式数学活动，实时监控学生进度
- 2023 年被 Amplify 收购，课堂工具整合为 Amplify Desmos Math

**商业模式与定价策略**
- 计算器工具完全免费
- Desmos Classroom/Amplify Desmos Math：面向学校/学区的 B2B 付费模式
- 隐私优先：使用 Matomo 自托管分析，禁用 Cookie

**用户规模与市场地位**
- 数学图形计算器领域的事实标准
- 北美几乎所有数学教师都知道 Desmos
- 被嵌入到大量第三方教育平台中（支持 iframe 嵌入）
- SAT/ACT 等标准化考试官方认可的计算器

**技术特点与体验**
- 纯 Web 应用，JavaScript SPA
- 极致流畅的交互体验
- 支持嵌入式部署（iframe API）
- 专注数学，不涉及物理仿真

**相对于 SeePhysics 的优劣势**
- 优势：数学可视化体验极致、免费、教师生态成熟、标准化考试认可
- 劣势：仅限数学（无物理）、无 AI、无 3D、无仿真引擎、无中文

#### 10. GeoGebra

**产品定位与核心功能**
- 动态数学软件平台，"将几何、代数、电子表格、图形、统计和微积分整合在一个引擎中"
- 六大工具：Calculator Suite（全能计算器）、Graphing Calculator、Geometry、3D Calculator、Scientific Calculator、Notes
- CAS 计算器：符号数学（导数、积分、因式分解）
- 课堂协作：实时学生进度监控，LMS 集成
- 社区资源：100 万+ 免费交互式教学资源（教师贡献）
- 考试模式：限制网络/软件访问
- JavaScript API：可嵌入外部网站

**商业模式与定价策略**
- 完全免费，开源软件（非商业用途）
- 2021 年加入 BYJU'S 家族，但保持独立运营
- 收入来源推测：机构许可、BYJU'S 生态协同

**用户规模与市场地位**
- 1 亿+ 全球用户
- 数学教育工具领域的全球领导者之一
- 覆盖 4-12 年级，支持数十种语言
- 数学引擎被数百个教育网站使用（包括 oPhysics）

**技术特点与体验**
- Next.js 构建的 Web 平台
- 有 3D 计算器功能（但非物理仿真）
- iOS/Android/Web 全平台
- 强大的社区 UGC 生态

**相对于 SeePhysics 的优劣势**
- 优势：免费开源、用户基数巨大、UGC 生态成熟、3D 数学可视化、全平台
- 劣势：专注数学而非物理、无 AI 生成、3D 仅限数学函数（非物理场景）、无课标对齐的物理内容、界面学习曲线较陡

### 四、中文市场竞品

#### 11. 物理大师

**产品定位与核心功能**
- 中国 K12 物理教育应用，主打"大片式"物理教学视频
- 核心特色：电影级制作质量的物理概念讲解视频
- 覆盖初中和高中物理课程，对齐中国课标
- 配套练习题和知识点梳理
- 移动端为主（iOS/Android）

**商业模式与定价策略**
- 免费增值模式
- 基础视频免费，高级内容和功能付费
- VIP 会员制：约 198-398 元/年
- 部分课程单独售卖

**用户规模与市场地位**
- 累计用户数千万级
- 在中国物理教育 App 中有一定知名度
- 但近年增长放缓，面临激烈竞争

**技术特点与体验**
- 视频为核心载体，非交互式
- 无仿真、无 AI、无 3D
- 移动端体验为主

**相对于 SeePhysics 的优劣势**
- 优势：中文原生、对齐中国课标、视频制作精良、品牌认知度
- 劣势：纯视频无交互、无 AI、无 3D 仿真、无自定义内容生成、产品形态较传统

---

#### 12. 洋葱学园（原洋葱数学）

**产品定位与核心功能**
- 中国 K12 在线教育平台，覆盖数学、物理、化学、语文、英语等多学科
- 核心特色：动画微课视频（5-8 分钟/节），趣味化讲解
- AI 自适应学习：根据学生水平推荐学习路径
- 配套练习、测试、错题本
- 教师端：备课工具、学情分析

**商业模式与定价策略**
- 免费增值模式
- 基础内容免费
- VIP 会员：约 300-500 元/年（按学科/年级）
- B2B 学校合作：智慧课堂解决方案

**用户规模与市场地位**
- 累计用户超过 5000 万
- 中国 K12 在线教育头部平台之一

## 第三部分：SeePhysics 差异化定位

### 核心价值主张

**"Ask anything. See the physics."——全球首个 AI 驱动的 3D 物理可视化生成平台。**

SeePhysics 不是又一个物理仿真库，也不是又一个 AI 聊天机器人。它占据了一个前所未有的交叉点：

```
传统仿真平台（PhET等）          AI教育工具（Khanmigo等）
    ↓ 预制内容，数量有限              ↓ 文字/对话式，无可视化
    ↓ 2D为主                        ↓ 无仿真能力
    ↓ 无法按需定制                   ↓ 无离线交付
              ↘                    ↙
           SeePhysics（交叉点）
           AI生成 + 3D可视化 + 零依赖HTML
           = 无限内容 × 沉浸体验 × 随处可用
```

### 市场空白分析

| 空白领域 | 说明 | SeePhysics 的填补方式 |
|---------|------|---------------------|
| AI + 3D 物理仿真 | 无竞品同时具备 | UPG 万物原理生成器 |
| 按需生成自定义实验 | PhET 等只有预制内容 | 用户输入任意问题即可生成 |
| 零依赖离线可视化 | 所有竞品需要网络或 App | 生成独立 HTML 文件可下载 |
| 中英双语 + 国际课标 | 中文平台不国际化，国际平台无中文 | 原生双语 + NGSS/GCSE/AP 对齐 |
| 可视化 + 公式 + 测验一体化 | 竞品通常只做其中一项 | 单个生成页面包含 3D 场景 + KaTeX 公式 + 交互控件 + 测验 |

### 独特竞争壁垒

1. **AI 生成引擎**：将自然语言问题转化为完整 3D 可视化页面的核心技术，竞品短期内难以复制
2. **零依赖 HTML 架构**：生成的页面无需任何外部依赖即可运行，这是一个独特的技术决策，使内容可以在任何环境（包括无网络的教室）中使用
3. **双语双课标**：同时服务中英文市场，对齐中国课标和国际课标（NGSS/GCSE/AP），这种组合在市场上独一无二
4. **内容飞轮效应**：每个用户生成的内容都可以丰富平台的内容库，形成网络效应

### 定位声明

> **For** 对物理充满好奇的学生、需要可视化教学工具的教师、以及想要直观理解物理原理的任何人，
> **SeePhysics is** 一个 AI 驱动的 3D 物理可视化平台，
> **that** 能将任何物理问题即时转化为包含 3D 交互场景、数学公式和知识测验的完整可视化页面，
> **unlike** PhET（预制有限仿真）、Khan Academy（文字/视频教学）、Wolfram Alpha（纯计算输出），
> **our product** 让物理从抽象公式变成可触摸的 3D 体验，并且生成的内容可以零依赖下载到任何设备离线使用。

### 最关键的三大风险及深度应对

#### 风险 1：AI 生成质量不稳定（严重）

**问题本质**：UPG 是产品核心，如果生成的 3D 可视化物理不准确、交互有 bug、或视觉效果差，用户信任会迅速崩塌。

**应对方案**：
1. **分层质量保障**
   - L1：AI 生成后自动运行 JavaScript 语法检查和 Three.js 渲染测试
   - L2：物理公式准确性校验（与 Wolfram Alpha API 交叉验证关键数值）
   - L3：高频主题（牛顿定律、抛体运动等）使用人工审核的"黄金模板"
2. **智能缓存与复用**
   - 相似问题（语义相似度 > 85%）直接返回已验证的缓存结果
   - 减少 AI 调用次数，同时保证质量
3. **用户反馈闭环**
   - 每个生成页面底部添加 "物理准确吗？" 的快速反馈按钮
   - 负面反馈自动触发人工审核队列
4. **渐进式质量提升**
   - 建立物理可视化质量评分模型
   - 持续收集数据优化生成 prompt

#### 风险 2：AI API 成本过高（严重）

**问题本质**：每次 UPG 生成都需要调用大语言模型 API，如果单次生成成本 > $0.10，在免费用户大量使用的情况下会快速烧钱。

**应对方案**：
1. **成本控制**
   - 积分制已经是好的开始（生成 10 积分，重新生成 5 积分）
   - 免费层限制（300 秒/天）有效控制免费用户成本
   - 优化 prompt 长度，减少不必要的 token 消耗
2. **智能缓存**
   - 建立语义缓存层：相似问题直接返回已生成结果的变体
   - 预估可减少 30-50% 的 API 调用
3. **模型策略**
   - 简单问题使用轻量模型（如 Claude Haiku），复杂问题使用高级模型
   - 探索微调专用模型降低推理成本
4. **成本监控**
   - 实时监控每用户、每生成的 API 成本
   - 设置成本预警阈值

#### 风险 3：PhET 等巨头添加 AI 功能（高）

**问题本质**：PhET 拥有 160+ 高质量仿真、全球知名度和学术背书。如果他们添加 AI 生成功能，SeePhysics 的差异化优势会被削弱。

**应对方案**：
1. **速度优势**：PhET 是学术项目，决策和开发周期长（通常 1-2 年），SeePhysics 需要在这个窗口期快速建立用户基础和品牌认知
2. **深耕差异化**：
   - 3D 可视化（PhET 以 2D 为主，转 3D 需要重写引擎）
   - 零依赖 HTML 下载（PhET 的架构不支持这种模式）
   - 中英双语（PhET 依赖社区翻译，质量参差不齐）
3. **建立网络效应**：用户生成的内容形成内容飞轮，这是 PhET 的预制模式无法复制的
4. **差异化定位**：不与 PhET 正面竞争"最全的仿真库"，而是定位为"AI 驱动的物理探索工具"

## 总结

SeePhysics 占据了一个真实且有价值的市场空白：**AI 生成 + 3D 物理可视化 + 零依赖离线交付**。这个组合在当前市场上没有直接竞品。

核心执行建议：
1. **冷启动聚焦"魔法时刻"**：让每个新用户都能在 30 秒内体验到 "输入问题 → 看到 3D 物理世界" 的震撼
2. **教师是关键杠杆**：一个教师 = 100+ 学生用户，教师满意度是增长的核心驱动力
3. **内容即获客**：每一篇 SEO 文章、每一个社交媒体帖子都应该嵌入可交互的物理可视化
4. **质量 > 数量**：宁可生成质量高的可视化让用户惊叹，也不要为了覆盖面牺牲准确性
5. **双语双市场**：中英文市场同步推进，但策略差异化——英文市场走社区/SEO，中文市场走 KOL/平台合作
