Concept
A string fixed at both ends can only sustain certain discrete frequencies — its normal modes. These occur when the string length equals an integer number of half-wavelengths, so nodes always fall at the fixed ends.
At each resonant frequency, the pattern of nodes (zero displacement) and antinodes (maximum displacement) is fixed. Higher harmonics have shorter wavelengths, more nodes, and higher frequencies that are exact integer multiples of the fundamental.
History: Marin Mersenne (1636) first quantified string vibrations. The nth harmonic has n antinodes and n+1 nodes (including endpoints).
Misconception: "Higher harmonics are louder." False — amplitude depends on how the string is driven, not mode number.
AP Link: Standing wave nodes count = n+1 (fixed-fixed); antinodes = n. This appears in AP Free Response regularly.
Why It Matters
  • • Musical instruments: string, wind, percussion all exploit harmonics
  • • MRI machines use resonant cavities to amplify signals
  • • Bridge design must avoid structural resonance frequencies (Tacoma Narrows)
