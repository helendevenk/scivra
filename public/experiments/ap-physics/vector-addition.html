Concepts
Vector Addition: Vectors add by combining their components. The resultant R = A + B can be found using the parallelogram rule or by adding x and y components separately: R_x = A_x + B_x, R_y = A_y + B_y.
Parallelogram Rule: Place both vectors at the same origin. Complete the parallelogram. The diagonal from the origin is the resultant vector. This is equivalent to the tip-to-tail (triangle) method.
