Concept

A thin lens bends all rays from a point object so they converge to — or appear to diverge from — a single image point. Three principal rays are enough to locate any image geometrically, without solving the wave equation.

Real images (dᵢ > 0) form where rays actually cross; virtual images (dᵢ < 0) form where backward-extended rays meet. Only real images can be projected on a screen. Diverging lenses (f < 0) always produce virtual, upright, reduced images of real objects.

Principal Rays
Ray 1 (Parallel): Enters parallel to axis → exits through focal point F₂
Ray 2 (Focal): Passes through focal point F₁ → exits parallel to axis
Ray 3 (Central): Passes through center of lens → undeviated
Key Insights
Common Misconception: When dₒ = f, the image is "at infinity" — the rays exit parallel, so they never converge to a finite image.
Sign Convention: Distances measured from the lens: dₒ > 0 always (real object); dᵢ > 0 (real image, opposite side); dᵢ < 0 (virtual, same side as object).
AP Connection: Know all four image cases: converging lens with dₒ > 2f, f < dₒ < 2f, dₒ < f; and diverging lens for any dₒ.
Why It Matters
  • Camera lenses: focus real inverted images on sensors
  • Eyeglasses: diverging or converging lenses correct vision defects
  • Microscopes/telescopes: compound lens systems for magnification
  • Projectors: converging lens projects real image on screen
