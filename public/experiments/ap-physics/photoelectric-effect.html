Conceptual Insight
Intensity vs Frequency
Intensity controls how many photons arrive per second — more photons → more electrons (if above threshold), but each electron gets the same energy.

Frequency controls how much energy each photon carries. Only frequency determines if electrons are ejected at all.
Classical Prediction (Wrong)
Classical wave theory predicted that any frequency should eject electrons if the wave is intense enough. Experiment proved otherwise — only frequency matters for ejection.
