Wave Interference AP Physics — Superposition Principle
Core Formulas
Concept
When two waves occupy the same space, their displacements add linearly — this is the Superposition Principle. The result is interference.
Constructive interference occurs where path difference Δr = nλ (whole number of wavelengths). Amplitudes add up to 2A.
Destructive interference occurs where Δr = (n + ½)λ. The waves cancel, leaving zero displacement.
Key Facts
Path Difference: Δr = r₁ − r₂. This determines the interference pattern, not the absolute distances.
Young's Double Slit: Same physics — bright fringes at d·sinθ = nλ, dark fringes at d·sinθ = (n+½)λ.
AP Connection: AP Physics 2 Unit 14 — Interference & Diffraction. Know constructive/destructive conditions, path difference, and fringe spacing.
Why It Matters
  • Noise-canceling headphones (destructive interference)
  • Optical coatings (anti-reflection films on lenses)
  • Radio signal dead zones from multi-path interference
