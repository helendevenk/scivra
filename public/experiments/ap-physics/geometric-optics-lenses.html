Concept

A two-lens system allows one lens's image to serve as the object for the next. The intermediate image position and size fully determines what the second lens receives.

For a compound microscope, both lenses converge the beam — the objective forms a magnified real image inside the barrel, then the eyepiece re-magnifies it. For a refracting telescope, a long-focal-length objective gathers light and the eyepiece magnifies the resulting image.

Real rays are traced through each lens using Snell's law in the paraxial approximation. Sign conventions: distances positive to the right.

Knowledge Cards

Historical Fact: Galileo improved the Dutch telescope in 1609 by pairing a converging and a diverging lens, achieving ~20× magnification.
Common Misconception: Total magnification is NOT f₁/f₂ for a two-lens system in general — that formula applies only to the afocal (telescopic) configuration.
AP Connection: AP Physics C Optics FRQs frequently ask for intermediate image position using 1/f = 1/do + 1/di applied twice.
