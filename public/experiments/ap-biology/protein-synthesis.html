Protein Synthesis Central Dogma: DNA → mRNA → Protein

Core Formulas

Central Dogma: genetic info flows DNA→RNA→Protein

Each codon (3 nucleotides) specifies one amino acid

Reading frame: 3-nt codons with no gaps or overlaps

Concept

Protein synthesis involves two key stages: transcription (DNA → mRNA in the nucleus) and translation (mRNA → protein at the ribosome). During translation, the ribosome reads mRNA in triplets called codons. Each codon recruits a specific tRNA carrying an amino acid. The start codon AUG always codes for methionine, while UAA, UAG, and UGA are stop codons that terminate the chain.

🔬 Key Fact
The genetic code is nearly universal — the same codons code for the same amino acids in bacteria, plants, and humans.
⚠️ Common Misconception
mRNA does NOT carry the same sequence as the template DNA strand — it matches the coding (non-template) strand, with T replaced by U.
